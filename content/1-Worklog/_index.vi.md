**Tuần 2:** [VPC Networking, Hybrid DNS, Route 53, VPC Peering & Transit Gateway](1.2-week2/)

**Tuần 3:** [EC2 cơ bản, Storage Gateway, S3 Static Website & CloudFront](1.3-week3/)

**Tuần 4:** [Amazon S3, Snow Family, Storage Gateway & Chiến lược DR](1.4-week4/)

**Tuần 5:** [IAM, Cognito, AWS Organizations, SSO & KMS](1.5-week5/)

**Tuần 6:** [IAM Security, Encryption, S3 Security, VPC Networking & DR](1.6-week6/)

**Tuần 7:** [Bài lab thực hành: VPC, EC2, Lambda, IAM, KMS & CloudTrail](1.7-week7/)

**Tuần 8:** [Database cơ bản: Keys, Indexing, RDBMS vs NoSQL, OLTP vs OLAP](1.8-week8/)

**Tuần 9:** [Data Pipeline: VPC/EC2/RDS, DMS/SCT, S3/Glue/Athena/QuickSight, Redshift](1.9-week9/)

**Tuần 10:** [Kiểm thử Backend: Lambda, API Endpoints & DynamoDB](1.10-week10/)

**Tuần 11:** [Kiểm thử Frontend: Components, UI/UX & API Integration](1.11-week11/)

**Tuần 12:** [Kiểm thử AI Module, Bedrock Integration & End-to-End Testing](1.12-week12/)
