# Nội dung nổi bật

## Amazon Q Developer (2:15 – 3:30 PM)

### Vòng đời phát triển hướng AI

- **Lập kế hoạch & Kiến trúc:** Thiết kế được AI hỗ trợ
- **Phát triển:** Tạo code từ ngôn ngữ tự nhiên
- **Kiểm thử:** Tự động tạo test
- **Triển khai:** Tạo Infrastructure as Code (IaC)
- **Bảo trì:** Tài liệu và hiện đại hóa

### Khả năng chính

- Tạo code với nhận thức ngữ cảnh
- Giải thích và tài liệu hóa code
- Hỗ trợ debug
- Tích hợp AWS:
  - Lambda
  - CloudFormation
  - CDK

### Demo trực tiếp

- Tạo REST API endpoint
- Giải thích thuật toán
- Tạo unit test
- Refactoring code
- Tạo code hạ tầng AWS

---

## Kiro AI Assistant (3:45 – 4:30 PM)

### Tính năng cốt lõi

- Hoàn thành code thông minh
- Hiểu đa file
- Trí tuệ cấp dự án
- Refactoring tự động
- Tạo tài liệu

### Cải thiện năng suất

| Chỉ số | Cải thiện |
|--------|-----------|
| Tốc độ viết code | Nhanh hơn **40%** |
| Giảm boilerplate | Giảm **60%** |
| Thời gian tài liệu | Giảm **50%** |
| Lỗi ban đầu | Ít hơn **30%** |

### Demo trực tiếp

- Hoàn thành code thông minh
- Refactoring đa file
- Phát hiện và sửa lỗi
- Tạo test
- Tự động hóa tài liệu

# Bài học rút ra

## Phát triển hướng AI

- AI biến đổi mọi giai đoạn của vòng đời phát triển
- Tự động hóa giải phóng developers cho công việc sáng tạo hơn
- Cải thiện năng suất đáng kể và có thể đo lường
- Chất lượng code cải thiện thông qua các patterns nhất quán

## Amazon Q Developer

- Mạnh mẽ cho phát triển tập trung AWS
- Ngôn ngữ tự nhiên sang code rất hiệu quả
- Bảo mật và best practices được tích hợp sẵn
- Hỗ trợ nhiều ngôn ngữ lập trình

## Kiro

- Xuất sắc trong việc hiểu cấp dự án
- Nhận thức ngữ cảnh mạnh cải thiện gợi ý
- Thao tác đa file tiết kiệm thời gian đáng kể
- Có thể tùy chỉnh theo tiêu chuẩn team

---

# Best Practices

- Cung cấp ngữ cảnh rõ ràng trong prompts
- Luôn review và hiểu code được tạo
- Lặp lại và tinh chỉnh gợi ý AI
- Duy trì nhận thức bảo mật mạnh
- Sử dụng công cụ AI để học và phát triển kỹ năng

# Áp dụng vào công việc

- Cài đặt **Amazon Q Developer** và **Kiro**
- Tích hợp cả hai công cụ vào quy trình phát triển hàng ngày
- Đo lường cải thiện năng suất theo thời gian
- Chia sẻ bài học và best practices với team

# Trải nghiệm sự kiện

Tham gia workshop **"AI-Driven Development Life Cycle"** đã cung cấp những hiểu biết có giá trị về cách AI đang cách mạng hóa phát triển phần mềm. Các demo thực hành về Amazon Q Developer và Kiro cho thấy các ứng dụng thực tế có thể ngay lập tức cải thiện năng suất.

Phiên làm việc nhấn mạnh rằng các công cụ AI không thay thế developers mà trao quyền cho họ tập trung vào giải quyết vấn đề cấp cao hơn trong khi tự động hóa các tác vụ lặp đi lặp lại.
